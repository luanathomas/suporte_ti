DESCRIÇÃO DO SISTEMA

O nome dos componentes do grupo: Andressa, Luana e Soraia

O tema escolhido: Sistema para Suporte de TI

Quais serão os dados registrados para cada caso: Em cada caso serão registrados:

Tipo de equipamento com problema (computador, notebook, impressora, etc…)

Sintoma principal (não liga, mensagem de erro, internet não funciona, etc…)

Sintoma secundário (exemplo: se a internet não funciona, abrirá um campo para selecionar o problema: não consegue acessar um site específico, cabo não funciona, etc…)

Qual será a técnica utilizada para recuperar casos similares: A técnica utilizada para recuperar os dados similares será através da Árvore K-D.

BANCO DE DADOS

O banco de dados pode ser encontrado no caminho ..\suporte_ti\db\suporte_ti.sql

ARQUIVO EXPLICATIVO SOBRE O SISTEMA

Para encontrar maiores informações sobre o sistema, é só clicar no link "Como utlizar a Central de Ajuda?" na página principal (index.php). Lá estão contidas informações sobre o funionamento do sistema para o administrador e para o usuário comum.
